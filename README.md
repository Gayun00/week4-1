## 🎮&nbsp;&nbsp;구현 사항

- 네비게이션: 메뉴 아이콘 클릭시 해당하는 페이지 위치로 이동
- 캐러셀: 3개 영역 로테이션 효과
- 배너 영역: 호버 시 줌인효과
- 푸터 : IE8 크로스 브라우징
- 팝업창: 클릭 시 외부 사이트 이동

### 👉 Git 커밋 컨벤션

> Option: 내용(상세 내용)

- Feat - 새로운 기능 추가
- Fix - 버그 수정
- Docs - 제품 코드 수정 없음
- Style - 코드 형식, 정렬, 주석 등의 변경
- Refactor - 코드 리팩토링
- Test - 테스트 코드 추가
- Chore - 환경설정, 빌드 업무, 패키지 매니저 설정등..
